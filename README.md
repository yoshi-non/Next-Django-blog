# Next.js × Django ブログ

- フロントエンド
  - Next.js
  - TypeScript
  - TailwindCSS
- バックエンド
  - Django
  - SQLite

## サーバ起動(VSCode)

バックエンド
```
cd django-api
.venv/Scripts/Activate.ps1
python manage.py runserver
```

フロントエンド
```
cd next-app
yarn start
```
